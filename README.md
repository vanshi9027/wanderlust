
# 🏡 Wanderlust 
Wanderlust is a full-stack web application inspired by Airbnb, where users can explore, create, edit, and manage property listings. This project demonstrates real-world backend development skills including authentication, CRUD operations, and MVC architecture.

## 📌 Features

* 🏠 Create, edit, and delete property listings
* 🔍 Browse all listings with details
* 📍 Location-based listings
* 📸 Image upload support
* 🔐 User Authentication (Login / Signup)
* ⭐ Reviews and Ratings system
* ⚡ Flash messages for user feedback
* 🛠 MVC architecture implementation

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/vanshi9027/wanderlust.git
```

### 2️⃣ Navigate to project folder

```
cd wanderlust
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Create `.env` file

Add your environment variables:

```
DB_URL=your_mongodb_connection
SECRET=your_secret_key
```

### 5️⃣ Run the server

```
node app.js
```

### 6️⃣ Open in browser

```
http://localhost:8080
```
